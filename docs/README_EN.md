This is SamHacker's personal tech blog, built with [Astro](https://astro.build/) and [TailwindCSS](https://tailwindcss.com/), integrated with [React](https://reactjs.org/), and using MDX for article content writing.

## üöÄ Deployment

After downloading the project source code, you can start the local development server with the following commands:

```bash
pnpm install
pnpm dev
```

For production builds, use the following commandsÔºö

```bash
pnpm build
pnpm preview
```

For detailed deployment information, refer to [DEPLOYMENT.md](DEPLOYMENT.md).

## ‚ú® Features

- [x] Built on Astro v5
- [x] Supports Markdown and MDX content writing
- [x] Fully designed for dark theme
- [x] Responsive image processing
- [x] Integrated shadcn/ui component library
- [x] Supports RSS and Sitemap generation
- [x] Built-in comment feature (Giscus)
- [x] Supports full-text search (Pagefind)
- [x] Article table of contents
- [x] Math formula rendering based on KaTeX
- [x] Enhanced code block display with Expressive Code Block
- [x] Real-time search suggestions based on Pagefind
- [x] Default support for Google Tag Manager
- [x] Featured 404 page

Developers interested in this project are welcome to participate. If you have any suggestions or issues, feel free to raise an Issue or Pull Request.

## üëÄ Version Requirements

- Node.js v22 or higher
- pnpm v9 or higher
